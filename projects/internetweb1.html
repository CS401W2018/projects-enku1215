Internet

Internet is a global network connects all computers and other electronic devices. Through connecting to the internet, we can research any information and reach out to anyone.

Web

Web is a software connects people and allows us to upload information and access others' information on the internet.

If you do not have an internet, you cannot access any web.

The difference between Internet and Web

Internet transport contents and web allows us to use those contents.

ASCII and Unicode

ASCII stands for American Standard Code for Information Interchange. It encodes all characters into a program using numbers. Each number represents a character. There are 128 characters.

Unicode is a global encoding standard for any writing systems and unique symbols from all around the world for any programs and systems.

UTF-8

What is UTF-8?

UTF-8 is an acronym for Unicode Transformation Format - 8 bit.
It is the most common and a broad encoding systems that you can represent any languages including ASCII characters and Unicode.

URL

Uniform Resource Locate or URL is the address that you use to access contents on internet. Any URL should consists of three things.
Protocols such as http or https, the domain name e.g. (example.com), and the path to a resource.

Domain

This is an address that represent your personal and business webpage on the internet. The domain owner can control their domains.
For example: "example.com"

HTTP vs HTTPS

HTTP and HTTPS are protocols that are used for transmitting the data between the website and your browser.

Question: Are they the same?

They are both communication protocols. The HTTP is for HyperText Transfer Protocol.It connects Web browser and Web server and communicates between them.
HTTPS is for HyperText Transfer Protocol Secure.The only difference is HTTPS is a secure version of HTTP.

IP Address

IP Address or Internet Protocol Address is a unique address that identifies any devices on the internet. It allows data to be sent between your devices and a web.

DNS

What is a DNS?

It is a Domain Name System which switches domain names into IP addresses. DNS helps browsers to locate and translate their internet domain names on the internet.

W3C Recommandations

W3C is for World Wide Web Consoritium which helps and set a standard for everyone who is building a web.

XML, XHTML, HTML, HTML5

XML or eXtensible Markup Language is a markup language made for keeping and transporting data. It is one of the W3C Recommandations.

XHTML (eXtensible HyperText Markup Language): XHTML is a mix of XML and HTML that follows strict rulebook for creating website.

HTML is HyperText Markup Language which is the standard language used to create and design webpages. It structures the content on the web by defining elements like headings, paragraphs, links, and images.

HTML5: This is the latest version of HTML, which includes new features and elements for today's web development, such as multimedia support (audio and video).
