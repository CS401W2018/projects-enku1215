Questions

Why don't we use the border property?- Once we use border, we cannot take it out. We will stick with a border. Also, border is an old design, if we want to have modern design, it is better to use CSS. It is more flexible to create a design you want.

What is the easiest way to color every other row a different color? - Using the following tag in CSS we can change the color of each row ".schedule tbody tr:nth-child(even or odd)", "nth-child()".

When should we use a table? - When we want to present clear data, we can use table. Tables make look your data easy to read for screen readers.
