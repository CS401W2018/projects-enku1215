Positioning

What is the difference between a block and an inline tag? Give examples of each one. Block tags like to take the whole space horizontally and text always start from a new line. Inline tags are opposite. It shares lines.

What is normal flow in HTML? Normal flow is without any changes, block and inlines are just the way they are.

What does float do? Why do you need to clear the float? - Float elements move the things in our page. It specifies how things should float.

What do absolute, fixed, and relative do? - Relative is an element that relates to its normal position. When we move it from where it normally is. Fixed - It will freeze the part.We usually use it for top parts like, headings and navigation. When you scroll the page, they won't move. Absolute - It removes the elements from the normal positions.

How can you make nice navigation buttons for mobile devices? - you can use larger buttons that are easy to tap, space them adequately, and design them with CSS tags like "UL" or "LI" and choose float.
