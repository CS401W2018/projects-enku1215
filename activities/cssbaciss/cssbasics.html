Explain the difference between inline, embedded, and external CSS. "Inline" - We can change the text color and text font using a style attribute within a html. "Embedded"- Same as inline but it uses style tags within the html head. "External CSS" - It is about using a separate "css" file linked to html using the link tag. We use it to separate the style and web design separate from the html. It makes it easier to manage the web. People who do the coding and create the design are usually separate in real life. Explain why you should only use external CSS. Because - Most webs created by programmers. The people who designs the web is separate. They can work only on the design from the document without the html file. What is a selector used for? - It is used for targetting specific HTML elements, such as the text within the main tag only or in the footer.
