1. What is the best way to make sure you get a good grade on the projects?

  • Be prepared for each class
    • To be prepared, we have to study before class starts on our own. For example, we can look up the topic for the day and assignments. If we have have any questions, we can ask them when we come to class. It will help us to be more engaging with the class.
  • Read and Follow the instructions
    • Each assignment has an instruction to help us do it right. Also there are rubrics that we can see how we will be graded on. By reading and following the instructions we will have good understanding about the assignment. With a clear and better undestanding it will be easier to work on them.
  • Don't wait until the due date
    • Each day check if there are any assignments. If there are any assignments, we can divide them into small steps and work on them everyday little by little. It will keep us from stress.
